# Disclaimer

## Who made this

This repository is a personal / educational game project created by a developer practicing web graphics, game loops, and interaction design.

## What this is

**WarpWing** is an original on-rails space shooter with a late-90s low-polygon aesthetic.

## What this is not

- Not an official Nintendo product
- Not Star Fox, Starwing, Lylat Wars, or any related title
- Not a ROM, emulator, asset dump, or recreation of copyrighted levels, models, audio, or story

## Why the disclaimer exists

Classic rail shooters defined a beloved genre. Homage to *genre conventions* (forward-scrolling combat, barrel rolls, rings) is common in indie games. Using another company's **trademarks, characters, or assets** is not. This project intentionally uses original branding and art so it can be shared as a learning portfolio piece without implying endorsement or ownership of anyone else's IP.
